# google-tasks-filter
Something I "vibe-coded" together to not be overwhelmed by tasks while reducing friction in my workflow

## Installation
1.  **Download the Source:**
    * Clone this repository
2.  **Open Chrome Extensions:**
    * Navigate to `chrome://extensions/` in your browser.
3.  **Enable Developer Mode:**
    * Toggle the switch in the top-right corner.
4.  **Load Extension:**
    * Click **"Load unpacked"** and select the folder containing this code.
5.  **Refresh:**
    * Open [tasks.google.com](https://tasks.google.com)
  
I might update this if I desire more functionality, but it's okay right now if you use the dedicated website
